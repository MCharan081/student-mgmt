# Student-Mgmt
# Description->
* This is a simple Student Database Management System built using Python.
* It provides a menu-driven interface to manage student records using CRUD operations.

# Features->
* Add new student records
* View all student records
* Update existing student details
* Delete student records
* Search for a student

# Technologies used->
* Python
* JSON (File Handling for Data Storage)

# Working->
* The program runs in a menu-driven format where users can choose options to perform different operations on student data.
